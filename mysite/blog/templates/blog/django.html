Installation

Downloads (Windows): https://www.python.org/downloads/windows
Downloads (Mac OS X): https://www.python.org/downloads/mac-osx/

System Environment Setting

One thing to watch out for: on the second screen of the installation wizard, marked "Customize", make sure you scroll down and choose the "Add python.exe to the Path" option, as shown here:

Python Environment Setup


The path details are handled by installer. One thing we need to do is to add Python director to path variable in system environment.

PYTHONPATH=/usr/local/bin/python3.4
After typing above commands in terminal, press Enter.

OS X

Verify the installation was successful by opening the Terminal application and running the python3 command:

$ python3 --version
Python 3.4.3

Set up Virtual environment

Virtual environment keeps your developing environment independent among different projects. The beauty of it is you don't need to worry about compatibility when you have one workstation with multiple projects working on a road.

Steps for Linux and OS X
  1. Create a virtual environment (also called a virtualenv)
    2. ~/SitePractise$ python3 -m venv myvenv
  2. Start your virtual environment by running
    3. ~/SitePractise$ source myvenv/bin/activate
  3. You will know that you have virtualenv started when you see that the prompt in your console looks like
    4. (myvenv) ~/SitePractise$
